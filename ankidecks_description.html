This deck uses the Wanikani method and data for the kanji and kanji-related Vocabulary from the Genki I 3rd Edition Book. The vocabulary-decks use sound files from Wanikani. Each Lesson has 5 decks:
  • 0 Radicals: Radicals are the components of kanji and should be learned first. (Or ignored if you don't like leaning Radicals)
  • 1 Kanjis: These are the kanjis in the order Genki wants you to learn them.
  • 2 Vocabulary: This deck contains all the vocabulary that is listed with the kanji and marked as important and some more I deemed important.
  • 3 Additional Vocabulary: This deck contains all the vocabulary from Wanikani that uses kanjis from this and previous lessons, but is not mentioned with the kanji in Genki.
  • 4 Unimportant Vocabulary: This deck contains all the vocabulary that is not marked as important by Genki. Contains words with kanji not mentioned in previous lessons. Some of the vocabulary in this deck might appear in later decks (2 or 3).
I recommend learning 0, 1 and 2 (and 3 if you have time). This deck was generated via python. Contributions welcome. Github link: https://github.com/mi-ael/genkikani Screenshots: Changes: 2021-08-01: Fixed vocabulary reading wrongfully being called Onyomi
